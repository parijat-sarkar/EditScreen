# Contributing to EditScreen

## Found a Bug?

1. Check [existing issues](https://github.com/parijat-sarkar/EditScreen/issues) first
2. [Open a new issue](https://github.com/parijat-sarkar/EditScreen/issues/new) with:
   - What you were doing
   - What happened
   - What you expected
   - Your screen data (anonymized if sensitive)
   - Screenshots

## Have a Feature Idea?

[Open an issue](https://github.com/parijat-sarkar/EditScreen/issues/new) with:
- The problem it solves
- How you'd use it
- Any examples

## Want to Contribute Code?

1. Fork the repo
2. Create a branch: `git checkout -b fix/your-fix-name`
3. Make your changes
4. Test thoroughly
5. [Submit a pull request](https://github.com/parijat-sarkar/EditScreen/pulls) with a clear description

## Code of Conduct

Be respectful. We're all here to make genome editing research easier.

## Questions?

Open an issue or email directly.

Thanks for contributing! 🧬
